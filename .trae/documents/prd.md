## 1. 产品概述

TalentX 是一个 AI Agent 驱动的人才试炼场平台。通过真实试炼场景（黑客松编程挑战）评估开发者/创业者的底层能力，自动生成六维特质画像并颁发三级能力认证。投资人可以通过画像发现真正有实力的早期创业者，不再依赖 pitch deck。
- 解决问题：AI 时代简历/pitch 不可信，传统面试看不准底层特质
- 目标用户：早期投资人/加速器（评估创业者）、开发者/创业者（证明自己）

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 候选人 | 邮箱注册 | 参加试炼、查看自己的画像和认证 |
| 投资人/HR | 邮箱注册 | 浏览候选人画像、搜索认证人才 |

### 2.2 功能模块
1. **首页(Landing)**: Hero 动画、产品介绍、核心价值主张、CTA 按钮
2. **试炼大厅**: 试炼卡片列表、难度标签、参与入口
3. **试炼进行中**: Agent 对话式引导、实时评估、代码提交流程
4. **个人画像**: 六维雷达图、维度详情、认证等级、能力标签
5. **认证证书页**: 认证展示、分享功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero 区域 | 动画背景 + 标语 + CTA 按钮，Claude 暖色风格 |
| 首页 | 产品介绍 | 三步走流程说明：选择试炼 → 完成挑战 → 获取画像 |
| 首页 | 数据展示 | 市场数据统计（AI 岗位增长、人才缺口等） |
| 试炼大厅 | 试炼卡片 | 展示可用试炼，含难度、类型、时长、参与人数 |
| 试炼进行中 | Agent 聊天 | 流式输出的对话界面，Agent 引导试炼流程 |
| 试炼进行中 | 提交面板 | 代码提交、进度追踪、实时评分反馈 |
| 个人画像 | 雷达图 | ECharts 六维雷达图（好奇心/靠谱/事实洁癖/多元思维/忍受不确定性/低ego高自驱） |
| 个人画像 | 维度详情 | 每个维度的行为指标和评分详情 |
| 个人画像 | 认证徽章 | C1/C2/C3 认证等级展示 |
| 认证证书页 | 证书展示 | 认证信息、有效期、验证二维码 |

## 3. 核心流程

用户进入平台 → 浏览试炼 → 选择黑客松试炼 → Agent 引导开始 → 完成编程挑战 → 提交代码 → AI 自动评审（EMA 五维评分）→ 生成六维画像 → 达标则颁发认证

```mermaid
flowchart TD
    "A[用户访问首页]" --> "B[浏览试炼大厅]"
    "B" --> "C[选择黑客松试炼]"
    "C" --> "D[Agent 引导开始]"
    "D" --> "E[完成编程挑战]"
    "E" --> "F[提交代码/PR]"
    "F" --> "G[AI 自动评审]"
    "G" --> "H[EMA 六维画像生成]"
    "H" --> "I{达标？}"
    "I" --> "是" --> "J[颁发认证证书]"
    "I" --> "否" --> "K[显示改进建议]"
    "J" --> "L[投资人可搜索查看]"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色：赤陶色 `#c96442`（品牌色/CTA）、珊瑚色 `#d97757`（链接/强调）
- 背景：羊皮纸暖色 `#f5f4ed`、象牙白 `#faf9f5`
- 文字：暖黑 `#141413`、橄榄灰 `#5e5d59`
- 字体：Playfair Display（标题）+ DM Sans（正文）
- 按钮风格：圆角、温暖色调、hover 时微妙放大
- 布局风格：编辑式排版、大量留白、不对称布局
- 图标风格：Lucide 图标库
- 阴影：Ring-based 阴影 `0px 0px 0px 1px`，非传统 box-shadow

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | Hero 区域 | Aurora/Silk 动画背景、SplitText 标题动画、Magnet CTA 按钮 |
| 首页 | 数据统计 | CountUp 数字动画、Staggered 入场 |
| 试炼大厅 | 试炼卡片 | SpotlightCard/TiltedCard、Stepper 流程指示 |
| 试炼进行中 | Agent 聊天 | AnimatedList 消息列表、打字机效果、流式输出 |
| 个人画像 | 雷达图 | ECharts 六维雷达图、CountUp 分数动画、StarBorder 认证徽章 |
| 认证证书页 | 证书 | 居中卡片、Noise 纹理背景、Border Glow 边框 |

### 4.3 响应式设计
- 桌面端优先（1440px 基准）
- 平板适配（768px）
- 移动端适配（375px）
- H5 响应式布局
